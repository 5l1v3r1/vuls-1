Install wth vulsctl on CentOS

Vulsctl

Vulsctl was created to ease setup. Each shell script is a wrapper for docker command.

setup Docker

  • Install Docker
  • Manage Docker as a non-root user

Clone Vulsctl

$ ssh centos@52.100.100.100  -i ~/.ssh/private.pem
$ sudo yum -y install git 
$ git clone https://github.com/vulsio/vulsctl.git
$ cd vulsctl

Fetch OVAL Database

$ ./oval.sh --redhat
$ ls -alh oval.sqlite3 
-rw-r--r--. 1 root root 11M Sep 12 06:28 oval.sqlite3

Fetch OVAL for the OS of the scan target server in advance. For details, see oval.sh and goval-dictionary

Fetch NVD Database

 ./cvedb.sh
$ ls -lh cve.sqlite3 
-rw-rw-r--. 1 root root 2.9G Sep 17 14:12 cve.sqlite3

cvedb.sh

Deploy gost

$ ./gost.sh --redhat
$ ls -lh gost.sqlite3 
-rw-r--r--. 1 root root 40M Sep 17 14:20 gost.sqlite3

For details, see gost.sh

Deploy go-exploitdb

$ ./exploitdb.sh
$ ls -lh go-exploitdb.sqlite3 
-rw-r--r--. 1 root root 17M Sep 17 14:22 go-exploitdb.sqlite3

Scan

Prepare config.toml in the same directory.

$ cat $HOME/vulsctl/config.toml 
[servers]
[servers.hostos]
host        = "10.10.10.10"
port        = "22"
user        = "centos"
# keypath in the Vuls docker container
keyPath     = "/root/.ssh/id_rsa"

$ ./scan.sh 
$ ./report.sh
$ ./tui.sh
